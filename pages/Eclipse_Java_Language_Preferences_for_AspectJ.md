We need Eclipse to use Java 5 syntax. To do so, do the following:
# Pull down the **Windows** menu
# Select **Preferences**
# Select **Java**
# Select **Compiler**
# Under JDK Compliance, set the **Java compliance level:** to 5.0.
# Click **OK**
